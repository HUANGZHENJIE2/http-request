# http-request
☞ http-request 是通过封装java httpURLConnection的 Maven 包，可快速使用java的GET、POST、PUT、DELETE，请求
## 介绍
小火箭是可以运行在Windows 平台的一个代理客户端，VMESS、VLESS、Trojan、SOCKET、Shadowscoks等主流代理协议。
